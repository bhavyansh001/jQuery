$ = jQuery

Syntax: $('selector').action()

'$(document).ready(function(){ jQuery code }' is used to execute jQuery when the document is loaded. Can also be written as $(function){ ... }

Selectors in jQuery: element (p), class(.class), id(#id)

* selects all elements $('*'); p.cl will select the p with class cl. p:first will select the first paragraph.

Events in jQuery

Mouse events

click, dblclick, mouseenter, mouseleave

Keyboard events

keypress, keydown, keyupevents

Form events

submit, change, focus, blur

window/document

load, resize, scroll, unload
